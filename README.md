## Возможности

- Канва с добавлением карточек разных типов (фотографии, заметки, локация, улика, др.)
- Возможность определения связей между карточками
- Локальное сохранение данных  
- Импорт и экспорт расследований (json)

## Технологии

- React + TypeScript  
- Vite  
- Tailwind CSS  
- React Flow  
- Zustand  

## Быстрый старт

```bash
npm install
npm run dev
```

## Сборка для прода
```bash
npm run build
```
